# 🤖 Smart Discord Chatbot

A smart and beginner-friendly Discord chatbot built using Python and discord.py.
This bot replies to user messages with predefined intelligent responses.

---

## ✨ Features
- Replies to 50+ common messages
- English language support
- Beginner friendly
- Secure token usage
- Easy to modify responses

---

## 🛠️ Technologies Used
- Python
- discord.py
- Replit
- GitHub

---

## 📂 Project Structure
smart-discord-chatbot/
- main.py
- requirements.txt
- .gitignore
- README.md

---

## 🚀 How It Works
The bot listens to messages in a Discord server.
When a message matches a keyword, the bot replies automatically.

---

## ▶️ How to Run
1. Install requirements  
   pip install -r requirements.txt  

2. Add your Discord bot token as environment variable  

3. Run  
   python main.py
